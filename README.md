# りもこズッ友キラキラポボモフォ2.0
日台技術交流のための、実験的なCLI翻訳・会話補助ツールです。
正式名称は「檸檬烯姐總合熱烈爆裂友情智慧板」。  
合言葉は「昨日的技術交流、今日的ズッ友」。
## 日本語
りもこズッ友キラキラポボモフォ2.0は、Computex Taipeiでの反省から生まれた、日本語話者と台湾の技術者がもっと自然に交流するためのCLIツールです。
日本語、ローマ字、英語、混合文で入力すると、相手に見せやすい自然な繁體中文・台灣華語へ変換します。相手側はスマホのWebUIから中文を入力でき、CLI側には自然な日本語訳が返ります。
手元のPCやノートPCに台灣華語入力環境がなくても、技術ブースやイベント会場で双方向に会話できることを目指しています。
### 背景
Computex Taipeiの現地で、英語の読み書きはできても、その場で細かい技術ニュアンスや温度感を口頭で伝えるのは難しい場面がありました。
日本語で考えたことを自然な台灣華語として相手に見せられたら、GPU、ROCm、Linux、OSS、ローカルLLMの話をもっと深く、もっと楽しくできるはずです。
このツールは、その反省から思いついて作り始めたものです。最初の版はおよそ3時間ほどで実装し、TwinkleAIブースで試しました。
しかし、最初の版はネットワークだけでなく、POST requestの処理やUI、会話の流れにも問題があり、現場では思ったように動きませんでした。Rustのコンパイルは通っていて、簡易テストでは動きそうに見えたため、少し早く現場投入してしまいました。当時付き合ってくださった方々には申し訳なく思っています。
その後、WebUI認証、会話バブル、相手ごとの表示、CLI履歴、WebUIとCLIの同期などを改善しました。まだ完成度は高くありませんが、必要な場面があれば、ぎりぎり試せるかもしれない小さな補助ツールとして公開しています。
### 名前について
`zuttomo` は日本語の「ずっと友だち」から来ています。
せっかくの出会いが言語の壁で少し惜しいものにならず、もっと自然に交流できて、長く友達になれる人が増えたらいいな、という願いを込めてこの名前にしました。
### 主な機能
- `/tw TEXT`, `/me TEXT`
  - 日本語・ローマ字・英語・混合文を、自然な台灣華語へ変換
- `/jp TEXT`, `/you TEXT`
  - 台灣華語・繁體中文を、自然な日本語へ変換
- `/pair`
  - 相手スマホ入力用WebUIを起動
- `/auth`
  - WebUI認証用の8桁コードを表示
- `/chat`
  - 会話モード
- `/question`
  - 技術質問モード
- `/model`
  - `mi25`, `openrouter`, `codex` のバックエンド切替
- 上下キーによるCLI入力履歴
### 使い方
開発環境で実行する場合:
```bash
cargo run -- --run
```

インストールして使う場合:

```bash
cargo install --path . --force
zuttomo
```

WebUIを使う流れ:

/pair
/auth

表示されたURLを相手に開いてもらい、/auth の8桁コードを入力してもらいます。

相手がWebUIに入力した中文は、CLI側に日本語訳として表示されます。CLI側で /tw や /me を使うと、相手側WebUIに台灣華語の発言として表示されます。

環境変数

.env.example を参考に .env を作成してください。

ZUTTOMO_DEFAULT_BACKEND=codex
ZUTTOMO_MI25_ENABLED=false
OPENROUTER_API_KEY=
ZUTTOMO_PAIR_HOST=0.0.0.0
ZUTTOMO_PAIR_PORT=8080
ZUTTOMO_PAIR_PUBLIC_URL=[your_domain_or_ip]

注意

このツールはまだ実験段階です。

翻訳品質、UI、通信処理、会話の流れには改善の余地があります。現時点では「完成した実用品」ではなく、イベントや技術交流の場で必要に応じて試せるかもしれない補助ツールです。

台灣華語

りもこズッ友キラキラポボモフォ2.0 是一個為了台日技術交流而做的實驗性 CLI 翻譯與會話輔助工具。

正式名稱是「檸檬烯姐總合熱烈爆裂友情智慧板」。
口號是「昨日的技術交流、今日的ズッ友」。

這個工具誕生於 Computex Taipei 現場的反省。它的目標是讓日文使用者與台灣的工程師，可以更自然、更輕鬆地互相交流。

使用者可以輸入日文、羅馬字、英文或混合句子，工具會把內容轉成適合直接給對方看的繁體中文、台灣華語。對方也可以用手機 WebUI 輸入中文，CLI 端會收到自然的日文翻譯。

即使手邊的 PC 或筆電沒有台灣華語輸入環境，也希望能在技術攤位或活動現場順利進行雙向對話。

背景

在 Computex Taipei 現場，我感受到即使可以讀寫英文，要在現場用口說即時傳達細節、技術脈絡和當下的感覺，還是很不容易。

如果能把日文裡想到的事情自然轉成台灣華語，就可以更深入、更開心地聊 GPU、ROCm、Linux、OSS、本地 LLM 等話題。

這個工具就是從那次反省開始的。我大概花了三個小時寫出第一版，然後在 TwinkleAI 攤位試用。

不過，最初版本不只是網路不穩，工具本身在 POST request 處理、UI、對話流程上也有問題，所以現場沒有像我想像中那樣順利運作。Rust 編譯已經通過，簡單測試時看起來也應該可以用，所以我有點太早拿到現場試用了。對當時願意陪我測試的人，我真的很不好意思。

之後我立刻改善了 WebUI 認證、對話泡泡、不同使用者的顯示、CLI 輸入歷史，以及 WebUI 與 CLI 的同步。它現在還不是很完整，也不敢說很實用，但如果有需要，也許可以作為一個小小的輔助工具試用看看。

關於名字

zuttomo 來自日文「ずっと友だち」，意思是「一直是朋友」。

我取這個名字，是希望難得的相遇不要因為語言隔閡而變得可惜。如果這個工具能讓更多人更容易交流，甚至成為長久的朋友，那就太好了。

主要功能

* /tw TEXT, /me TEXT
    * 將日文、羅馬字、英文或混合句子轉成自然的台灣華語
* /jp TEXT, /you TEXT
    * 將台灣華語、繁體中文轉成自然的日文
* /pair
    * 啟動給對方手機使用的 WebUI
* /auth
    * 顯示 WebUI 認證用的 8 位數字
* /chat
    * 對話模式
* /question
    * 技術問題模式
* /model
    * 切換 mi25, openrouter, codex 後端
* CLI 可用上下鍵瀏覽輸入歷史

使用方式

在開發環境執行:

```bash
cargo run -- --run
```

安裝到本機:

```bash
cargo install --path . --force
zuttomo
```

使用 WebUI:

/pair
/auth

請對方打開顯示的 URL，並輸入 /auth 顯示的 8 位數字。

對方在 WebUI 輸入的中文會在 CLI 端顯示成日文翻譯。CLI 端使用 /tw 或 /me 時，台灣華語內容會顯示在對方的 WebUI 上。

環境變數

請參考 .env.example 建立 .env。

ZUTTOMO_DEFAULT_BACKEND=codex
ZUTTOMO_MI25_ENABLED=false
OPENROUTER_API_KEY=
ZUTTOMO_PAIR_HOST=0.0.0.0
ZUTTOMO_PAIR_PORT=8080
ZUTTOMO_PAIR_PUBLIC_URL=[your_domain_or_ip]

注意

這個工具目前還在實驗階段。

翻譯品質、UI、通訊處理和對話流程都還有很多可以改善的地方。現在它不是一個完成度很高的正式產品，而是希望在活動或技術交流場景中，必要時可以試用看看的一個輔助工具。
