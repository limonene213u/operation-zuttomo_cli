# りもこズッ友キラキラポボモフォ2.0

## 日本語

りもこズッ友キラキラポボモフォ2.0は、Computex Taipeiでの反省から生まれた、日本人技術者と台湾の現地技術者がもっと簡単に相互交流するためのCLIツールです。

正式名称は「檸檬烯姐總合熱烈爆裂友情智慧板」。合言葉は「昨日的技術交流、今日的ズッ友」です。

日本語、ローマ字、英語、混合文で入力すると、相手に見せやすい自然な繁體中文・台灣華語へ変換します。相手側はスマホのWebUIから中文を入力でき、CLI側には自然な日本語訳が返ります。ThinkPad側に台灣華語入力環境がなくても、技術ブースで双方向に会話できることを目指しています。

### 背景

Computex Taipeiの現地で、英語だけに頼ると細かい技術交流や温度感の共有が難しい場面がありました。日本語で考えたことを自然な台灣華語として相手に見せられたら、GPU、ROCm、Linux、OSS、ローカルLLMの話をもっと深く楽しくできるはずです。

このツールはその反省から、思いついてからおよそ3時間ほどで最初の実装を書き、TwinkleAIブースで試しました。けれど、最初の版は接続やUI、会話の流れがうまく噛み合わず、現場で付き合ってくれた方々には申し訳ないことをしました。

その後すぐに、WebUI認証、会話バブル、相手ごとの表示、CLI履歴、WebUIとCLIの同期などを改善しました。あの場で試してくれた人たち、待ってくれた人たち、そして技術交流に付き合ってくれたみなさんに感謝します。

### 主な機能

- `/tw TEXT`, `/me TEXT`: 日本語・ローマ字・英語を自然な台灣華語へ変換
- `/jp TEXT`, `/you TEXT`: 台灣華語・繁體中文を自然な日本語へ変換
- `/pair`: 相手スマホ入力用WebUIを起動
- `/auth`: WebUI認証用の8桁コードを表示
- `/chat`: 会話モード
- `/question`: 技術質問モード
- `/model`: `mi25`, `openrouter`, `codex` のバックエンド切替
- 上下キーによるCLI入力履歴

### 使い方

```bash
cargo run -- --run
```

インストールする場合:

```bash
cargo install --path . --force
zuttomo
```

WebUIを使う流れ:

```text
/pair
/auth
```

表示されたURLを相手に開いてもらい、`/auth` の8桁コードを入力してもらいます。相手がWebUIに入力した中文はCLI側に日本語訳として表示されます。CLI側で `/tw` や `/me` を使うと、相手側WebUIに台灣華語の発言として表示されます。

### 環境変数

`.env.example` を参考に `.env` を作成してください。

```text
ZUTTOMO_DEFAULT_BACKEND=codex
ZUTTOMO_MI25_ENABLED=false
OPENROUTER_API_KEY=
ZUTTOMO_PAIR_HOST=0.0.0.0
ZUTTOMO_PAIR_PORT=8080
ZUTTOMO_PAIR_PUBLIC_URL=https://zuttomo-test.aets-hiroshima.org
```

## 台灣華語

りもこズッ友キラキラポボモフォ2.0 是一個從 Computex Taipei 現場反省中誕生的 CLI 工具。它的目標是讓日本工程師與台灣現地工程師可以更簡單、更自然地互相交流。

正式名稱是「檸檬烯姐總合熱烈爆裂友情智慧板」。口號是「昨日的技術交流、今日的ズッ友」。

使用者可以輸入日文、羅馬字、英文或混合句子，工具會把內容轉成適合直接給對方看的繁體中文、台灣華語。對方也可以用手機 WebUI 輸入中文，CLI 端會收到自然的日文翻譯。即使 ThinkPad 上沒有台灣華語輸入環境，也希望能在技術攤位上順利雙向對話。

### 背景

在 Computex Taipei 現場，我感受到如果只依靠英文，有時很難把細節、技術脈絡和當下的感覺完整傳達出來。如果能把日文裡想到的事情自然轉成台灣華語，就可以更深入、更開心地聊 GPU、ROCm、Linux、OSS、本地 LLM 等話題。

這個工具就是從那次反省開始的。我大概花了三個小時寫出第一版，然後在 TwinkleAI 攤位試用。不過，最初版本在連線、UI、對話流程上都沒有順利運作。對當時願意陪我測試的人，我真的很抱歉。

之後我立刻改善了 WebUI 認證、對話泡泡、不同使用者顯示、CLI 輸入歷史，以及 WebUI 與 CLI 的同步。謝謝當時願意測試、願意等待，也願意跟我進行技術交流的大家。

### 主要功能

- `/tw TEXT`, `/me TEXT`: 將日文、羅馬字、英文轉成自然台灣華語
- `/jp TEXT`, `/you TEXT`: 將台灣華語、繁體中文轉成自然日文
- `/pair`: 啟動給對方手機使用的 WebUI
- `/auth`: 顯示 WebUI 認證用的8位數字
- `/chat`: 對話模式
- `/question`: 技術問題模式
- `/model`: 切換 `mi25`, `openrouter`, `codex` 後端
- CLI 可用上下鍵瀏覽輸入歷史

### 使用方式

```bash
cargo run -- --run
```

安裝到本機:

```bash
cargo install --path . --force
zuttomo
```

使用 WebUI:

```text
/pair
/auth
```

請對方打開顯示的 URL，並輸入 `/auth` 顯示的8位數字。對方在 WebUI 輸入的中文會在 CLI 端顯示成日文翻譯。CLI 端使用 `/tw` 或 `/me` 時，台灣華語內容會顯示在對方的 WebUI 上。
